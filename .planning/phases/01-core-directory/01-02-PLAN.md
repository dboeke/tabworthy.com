   - `getCategories()`: Returns all categories ordered by displayOrder, with count of channels (via channelCategories + channelTopics).
   - `getCategoryBySlug(slug: string)`: Returns category with its topics (ordered by displayOrder) and channels directly assigned to the category via channelCategories join table. Include channel's creator, platform, and tags.
   - `getTopicBySlug(categorySlug: string, topicSlug: string)`: Returns topic with its parent category, channels (with creator, platform, tags), and all tags used by channels in this topic (for the filter bar).
   - `getChannelsForTopic(topicId: string, tagSlugs?: string[])`: Returns channels filtered by tag slugs (AND logic -- channel must have ALL specified tags). Include creator, platform, tags. If no tagSlugs, return all channels for topic.
   - `getChannelsForCategory(categoryId: string)`: Returns channels directly assigned to category via channelCategories.
   - `getAllTagsForTopic(topicId: string)`: Returns unique tags used by channels in a topic.

   Use Drizzle relational queries (`db.query.channels.findMany({ with: { creator: true, platform: true, ... } })`).

   For AND tag filtering: query channels that have ALL specified tags. Use a subquery or filter approach:
