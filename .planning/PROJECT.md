# Tabworthy

## What This Is

A human-curated channel guide for online video — a modernized Yahoo Directory for the creator economy. Tabworthy helps people discover the best creators across YouTube, Twitch, Patreon, Nebula, Substack, and other platforms by browsing an editorially curated, topic-organized directory with rich channel profiles and community nominations.

## Core Value

People can browse by topic and find creators worth following — curated by humans, not algorithms.

## Requirements

### Validated

(None yet — ship to validate)

### Active

- [ ] Two-level taxonomy browsing (Category > Topic) with tag-based filtering within topics
- [ ] Editorially curated channel listings with rich profiles (description, best videos, platform links, related channels)
- [ ] Community nomination and voting system for new channels/categories
- [ ] User accounts that unlock personalization, favorites, and lists
- [ ] Platform-agnostic design supporting open-ended platform additions
- [ ] Public browsing without account requirement
- [ ] Search across channels, categories, and topics
- [ ] Wireframes-ready feature specifications for creative team handoff

### Out of Scope

- Monetization / revenue features — passion project for v1
- Content aggregation / embedded video playback — Tabworthy is the directory, not the player
- Mobile native apps — web-first
- Real-time chat or social features beyond voting/nominations
- Creator self-service / claiming profiles — editorial-led for now

## Context

- Inspired by the old Yahoo Internet Directory: a browsable, human-organized hierarchy
- Editorial team curates the core directory; community can nominate and vote on additions
- Nominations could appear ranked at the bottom of category pages or in a dedicated section
- Supports any video/content platform — YouTube, Twitch, Patreon, Nebula, Substack are the initial set
- Primary output of this project phase is feature specs detailed enough for designers to create wireframes

## Constraints

- **Output**: Feature specifications and planning docs — creative team handles UI/UX design
- **Curation model**: Editorial-first with community input — not open submission
- **Platform scope**: Must be platform-agnostic from the start — no hard-coding to specific platforms
